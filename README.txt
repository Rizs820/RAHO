NOTES:-
1. Database File is included in database folder.
2. Some APIs are used as trial version (for demo purpose only) so they will stop working after 90 days(vide 20-3-18). 
3. This PHP code is only used for JSON data transmission and API fetch purpose.
4. All the respective folders are having the respective Module APIs.

Installation/Hosting:-
1. Install WAMP/XAMPP/AMPPS (Having PHP>=5.6).
2. Paste the PHP_Code folder in www/htdocs folder of your server.
3. Open PHPMyAdmin create database and import file "database/raho.sql".
4. Make the changes in dbConnect file in "includes/dbConnect.inc.php".
5. Now you can use this APIs in your android API by using your IP Address (or domain).
